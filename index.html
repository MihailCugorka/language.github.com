! The Last of the Phoenicians

Many writing systems that we use today descended directly from the "abjad" alphabet created by Phoenicians in 1050 BC.

  • alf ("ox") ʾ [ʔ] => alpha Αα (1) => Aa
  • bet ("house") b [b] => beta Ββ (2) => Bb
  • gaml ("camel") g [ɡ] => gamma Γγ (3) => Gg, Cc
  • delt ("door") d [d] => delta "Δδ(4)" => Dd
  • he ("window") h [h] => epsilon "Εε(5)" => Ee
  • wau ("hook") w [w] => upsilon "Υυ(400)", digamma "Ϝϝϛ(6)" => Ww Yy Uu Vv, Ff
  • zai ("weapon") z [z] => zeta "Ζζ(7)" => Zz
  • het ("wall") ḥ [ħ] => eta "Ηη(8)" => Hh
  • tet ("wheel") ṭ [tˤ] => theta "Θθ(9)"
  • yod ("hand") y [j] => iota "Ιι(10)" => Ii, Jj
  • kaf ("palm") k [k] => kappa "Κκ(20)" => Kk
  • lamd ("goad") l [l] => lambda "Λλ(30)" => Ll
  • mem ("water") m [m] => mu "Μμ(40)" => Mm
  • nun ("serpent") n [n] => nu "Νν(50)" => Nn
  • semk ("fish") s [s] => xi "Ξξ(60)", chi "Χχ(600)" => Xx
  • ain ("eye") ʿ [ʕ] => omicron "Οο(70)", omega "Ωω(800)" => Oo
  • pe ("mouth") p [p] => pi "Ππ(80)" => Pp
  • sade ("hunt") ṣ [sˤ] => san "Ϻϻ"
  • qof ("needle head") q [q] => koppa "Ϙϙϟ(90)", phi "Φφ(500)", psi "Ψψ(700)" => Qq
  • rosh ("head") r [r] => rho "Ρρ(100)" => Rr
  • shin ("tooth") š [ʃ] => sigma "Σσς(200)" => Ss
  • tau ("mark") t [t] => tau "Ττ(300)" => Tt
