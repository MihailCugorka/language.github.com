The Last of the Phoenicians

Many writing systems that we use today descended directly from the "abjad" alphabet created by Phoenicians in 1050 BC.

  • alf ("ox") ʾ [ʔ] => alpha Αα (1)
  • bet ("house") b [b] => beta Ββ (2)
  • gaml ("camel") g [ɡ] => gamma Γγ (3) - g
  • delt ("door") d [d] => delta "Δδ(4)"
  • he ("window") h [h] => epsilon "Εε(5)"
  • wau ("hook") w [w] => upsilon "Υυ(400)"
  • zai ("weapon") z [z] => zeta "Ζζ(7)"
  • het ("wall") ḥ [ħ] => eta "Ηη(8)"
  • tet ("wheel") ṭ [tˤ] => theta "Θθ(9)"
  • yod ("hand") y [j] => iota "Ιι(10)"
  • kaf ("palm") k [k] => kappa "Κκ(20)"
  • lamd ("goad") l [l] => lambda "Λλ(30)"
  • mem ("water") m [m] => mu "Μμ(40)"
  • nun ("serpent") n [n] => nu "Νν(50)"
  • semk ("fish") s [s] => xi "Ξξ(60)", chi "Χχ(600)"
  • ain ("eye") ʿ [ʕ] => omicron "Οο(70)", omega "Ωω(800)"
  • pe ("mouth") p [p] => pi "Ππ(80)"
  • sade ("hunt") ṣ [sˤ] => san "Ϻϻ"
  • qof ("needle head") q [q] => koppa "Ϙϙϟ(90)", phi "Φφ(500)", psi "Ψψ(700)"
  • rosh ("head") r [r] => rho "Ρρ(100)"
  • shin ("tooth") š [ʃ] => sigma "Σσς(200)"
  • tau ("mark") t [t] => tau "Ττ(300)"
