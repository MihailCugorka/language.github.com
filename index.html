The Last of the Phoenicians

Many writing systems that we use today descended directly from the "first alphabet". An alphabet with no vowels is known as an "abjad".

  • alf ("ox") => alpha Αα (1) ...represented a glottal stop ("ʔ" in dene)
  • bet ("house") => beta Ββ (2)
  • gaml ("camel") => gamma Γγ (3) - g
  • delt ("door") => delta "Δδ(4)"
  • he ("window") => epsilon "Εε(5)"
  • wau ("hook") => upsilon "Υυ(400)"
  • zai ("weapon") => zeta "Ζζ(7)"
  • het ("wall") => eta "Ηη(8)"
  • tet ("wheel") => theta "Θθ(9)"
  • yod ("hand") => iota "Ιι(10)"
  • kaf ("palm") => kappa "Κκ(20)"
  • lamd ("goad") => lambda "Λλ(30)"
  • mem ("water") => mu "Μμ(40)"
  • nun ("serpent") => nu "Νν(50)"
  • semk ("fish") => xi "Ξξ(60)", chi "Χχ(600)"
  • ain ("eye") => omicron "Οο(70)", omega "Ωω(800)"
  • pe ("mouth") => pi "Ππ(80)"
  • sade ("hunt") => san "Ϻϻ"
  • qof ("needle head") => koppa "Ϙϙϟ(90)", phi "Φφ(500)", psi "Ψψ(700)"
  • rosh ("head") => rho "Ρρ(100)"
  • shin ("tooth") => sigma "Σσς(200)"
  • tau ("mark") => tau "Ττ(300)"
